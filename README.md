# .claude

Org-shared **Claude Code** configuration and skills for the `getdue-dev` polyrepo.

- **Type:** Org config (analogous to [`.github`](https://github.com/getdue-dev/.github)).
- **Publishes:** nothing — no app build, no container image, no package. It holds
  shared Claude Code skills, settings, and prompts consumed by engineers across the
  GetDue repositories.

## CI

This is a configuration-only repo with no build artifact, so it carries no reusable CI
pipeline. `main` is protected and changes land via PR.

## Conventions

Repository conventions follow
[`getdue-docs/engineering/01-repositories.md`](https://github.com/getdue-dev/getdue-docs/blob/main/engineering/01-repositories.md).
